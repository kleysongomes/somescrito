SOM ESCRITO

O SITE

O Som Escrito é um site desenvolvido por pessoas para pessoas, tendo o foco no publico sudo.

OBJETIVO

Nosso objetivo é informar para pessoas surdas o som que as coisas emitem, se emitem e qual a densidade sonora.

IDEIA

A ideia surge para pessoas surdas conseguirem saber se o objeto utilizado emite um som e qual a emoção que esse som garrega.

Usando o Site

Como funciona a usabilidade de nosso site?
Você vai ter acesso a uma relação de emissores de som, nesta relação vocês terá alguns indicadores como se o item emite som, qual a emoção que esse som trás em outras pessoas, e qual a densidade sonora desse som ou seja o quando ele pode incomodar ou não alguém.
